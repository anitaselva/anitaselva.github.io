Group project | 2023 | One month

42sh

For this project we had to recreate a bash-like terminal in C. During this project we learned how to do a proper lexing and parsing, we also re-implemented some built-ins. We had to manage command execution, variable expansion, subshells, command substitutions and some other bash features. We used git for versionning as usual and did a python test suite.
