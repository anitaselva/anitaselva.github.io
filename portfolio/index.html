EPITA big projects

42sh

42sh was a one month project in group of four during the 3rd year of EPITA. We had to recreate a bash-like terminal in C. During this project we learned how to do a proper lexing and parsing, we also re-implemented some built-ins. We had to manage command execution, variable expansion, subshells, command substitutions and some other bash features. We used git for versionning as usual and did a python test suite.

OCR

OCR was a one month project in group of four during the 2nd year of EPITA. We created a program for sudoku grid recognition and solving in C. The main parts of the projects were the filtering of the image to make the detection easier, the grid detection, an AI for character recognition, the sudoku solver and the user interface to filter detect and solve separately or at once. The project versionning was done via git.

NYR: the last stand

NYR was a six month project in group of four during the 1st year of EPITA. The project consisted in creating a game in C# using Unity. Mainly inspired by Assassin's Creed: Valhalla and Project winter our game was about a viking that has to save his island. We had to implement AI, Map and UI design, Saving, Multiplayer and finally Items and Player management. All of our work is documented in a website. Click here to see more!
